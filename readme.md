# cptm-status

> A real-time API for [CPTM](https://www.cptm.sp.gov.br/) status

## Install

```bash
$ yarn add cptm-status
```

## Usage

```js
const cptmStatus = require('cptm-status')

cptmStatus()
```

## API

### cptmStatus()

returns a `promise`
