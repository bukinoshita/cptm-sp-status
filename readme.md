# cptm-sp-status

> A real-time API for [CPTM](https://www.cptm.sp.gov.br/) status

## Install

```bash
$ yarn add cptm-sp-status
```

## Usage

```js
import { cptmStatus } from 'cptm-sp-status'

cptmStatus()
```

## API

### cptmStatus()

returns a `promise`
